cdInputWrapper
directive in module cd.ui

Description

Wrap input element in form

Usage

as attribute
<ANY cd-input-wrapper
     [container-field-class="{String}"]
     [container-input-class="{String}"]
     [error-max-length="{String}"]
     [error-number="{String}"]
     [error-required="{String}"]
     [label-class="{String}"]
     [label-title="{String}"]
     [name="{String}"]
     [required]>
   ...
</ANY>
