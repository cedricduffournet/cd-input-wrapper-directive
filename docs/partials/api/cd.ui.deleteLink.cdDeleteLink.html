cdDeleteLink
directive in module cd.ui

Description

Create en link with a angular-strap popover and angular-strap tooltip Based on angular-strap

Usage

as element:
<cd-delete-link
       [action-data-return="{Object}"]
       [animation="{String}"]
       [content="{String}"]
       [emit-name="{String}"]
       [label-btn-action="{String}"]
       [label-btn-cancel="{String}"]
       [title-popover="{String}"]
       [title-tooltip="{String}"]>
</cd-delete-link>

Parameters

ParamTypeDetails
actionDataReturn
(optional)
Object

data to return on action click

animation
(optional)
String

apply a CSS animation powered by ngAnimate

content
(optional)
String

angular-strap popover content

emitName
(optional)
String

Event name to emit on action click

labelBtnAction
(optional)
String

label for action button

labelBtnCancel
(optional)
String

label for cancel button

titlePopover
(optional)
String

angular-strap popover title

titleTooltip
(optional)
String

angular-strap tooltip title
